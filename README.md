# literature-rename

A Codex skill for batch-renaming academic literature files by document title.

It supports PDF and DOCX files, translated and bilingual copies, conservative rename planning, dry-run execution, Windows case-only renames, and action logs.

## Included

- `SKILL.md`
- `scripts/build_rename_plan.py`
- `scripts/apply_rename_plan.py`
- `agents/openai.yaml`
